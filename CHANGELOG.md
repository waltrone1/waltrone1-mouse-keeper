## [1.0.0.0] - 2026-05-06

### Added

- Initial public release of waltrone1 Mouse-Keeper
- Portable Windows utility for keeping a PC active during long-running tasks
- Small desktop interface for quick start and pause workflow
- Configurable activity interval
- Configurable mouse movement distance
- Optional auto-click function
- Auto-click disabled by default
- Always-on-top option
- Transparency setting
- Multiple color themes
- Settings window
- WALTRONE mascot / bunny branding
- Application icon
- Website link
- YouTube link
- Support link
- Local configuration handling
- Useful for uploads, exports, render jobs, presentations, monitoring workflows and similar long-running tasks
- py2exe build files for creating a Windows executable
- GitHub repository structure with README, LICENSE, CHANGELOG, screenshots and documentation folders

### Notes

- waltrone1 Mouse-Keeper is a small helper tool for local Windows use.
- The tool is intended to reduce interruptions caused by standby, idle state or screensaver behavior.
- The optional auto-click function should only be enabled when it is really needed.
- Auto-click should only be used in situations where automated clicking is appropriate and allowed.
- Some company or managed IT environments may restrict tools that simulate activity.
- Use responsibly and only on systems where you are allowed to run such tools.
